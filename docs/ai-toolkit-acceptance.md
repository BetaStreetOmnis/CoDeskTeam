# AI 工具箱 100% 验收矩阵

## 范围

本清单对应 Excel 中的 `AI+通用工具集（云端、边缘端均需支持）`，并对当前前端卡片里已经承诺的能力做可验收化约束。

## 视觉设计与图像处理

- 图片编辑与裁剪
  - 支持上传一张或多张图片
  - 支持按预设比例批量裁剪：`1:1`、`4:3`、`16:9`
  - 支持统一宽度缩放
  - 支持文字水印叠加
- 图像增强与滤镜
  - 支持基础增强：锐化、对比度、饱和度、亮度
  - 支持预设滤镜：商务、清晰、暖色、戏剧化
  - 支持透明背景导出
- 格式转换
  - 支持输出 `png`、`jpg`、`webp`、`avif`
  - 支持多文件批量输出

## 视频与音频处理

- 视频剪辑与合并
  - 支持按秒裁剪预览片段
  - 支持多视频合并
  - 支持封面抽帧
  - 支持字幕烧录
- 视频转码与分发
  - 支持 `H.264` / `H.265`
  - 支持 `1080p` / `720p` / `480p`
  - 支持 `MP4` 与 `HLS`
- 音频降噪与编辑
  - 支持降噪与响度标准化
  - 支持静音裁剪
  - 支持导出 `MP3`

## 内容创作与写作

- 多场景文案生成
  - 支持多版本文案生成
  - 支持多语输出
  - 支持品牌词约束
- 排版优化与改写
  - 支持对输入文稿进行结构重排
  - 支持审阅建议
  - 支持版本摘要
- 文档创建与编辑
  - 支持生成方案文档
  - 支持生成 PPT
  - 支持生成报价单

## 办公与效率提升

- 日程管理
  - 支持录入日程项
  - 支持冲突检测
  - 支持提醒策略生成
  - 支持导出 `ICS`
- 任务协作
  - 支持任务拆解与 CSV 看板导出
  - 支持协作执行手册输出
  - 支持协作原型输出
- 文件转换与格式处理
  - 支持文本/表格/图片/音视频常见格式转换
  - 支持转换结果清单
  - 支持失败原因记录

## 验收口径

- 前端：`/api/skills` 返回的默认 payload 能配置上述能力
- 后端：`/api/skills/pipeline/*` 能真实执行并返回下载产物
- 产物：输出文件注册到 `file_records`，可经 `/api/files/{file_id}` 下载
- 报告：每条 pipeline 至少生成一份执行报告，列出成功产物与告警
